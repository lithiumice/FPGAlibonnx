GIT使用分析：

1.在你的电脑上还没有装过GIT的时候先安装GIT

2.在GIT_BASH中先执行：ssh-keygen -t rsa -C "1700695611@qq.com"

3.将.ssh文件中的东西copy到github中

4.然后回到文件： git remote add origin git@github.com:Yueyang123/UcOSII.git

5.git add

6.git commit 

7git push(第一次： git push --set-upstream origin master)
